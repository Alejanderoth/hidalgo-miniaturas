# TFG - Sistema de Digitalización para Hidalgo Miniaturas

## Descripción

Este proyecto consiste en el desarrollo de una aplicación web para la digitalización de la empresa familiar Hidalgo Miniaturas.
El sistema permitirá gestionar productos, pedidos, clientes y procesos internos de producción, sustituyendo el sistema actual basado en correos electrónicos y gestión manual.
El objetivo es mejorar la eficiencia, reducir errores y centralizar toda la información en una única plataforma.

## Tecnologías utilizadas

- PHP (Laravel
- MySQL
- Apache (XAMPP)
- HTML, CSS, JavaScript
- Git y GitHub
